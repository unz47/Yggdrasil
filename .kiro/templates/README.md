# テンプレート使用ガイド

## 📁 テンプレート一覧

### アイテム系
- **item_normal.md** - 通常アイテム（素材、装飾品など）
- **item_tool.md** - ツール（剣、ピッケル、斧、シャベル、クワ）
- **item_armor.md** - 防具（ヘルメット、チェスト、レギンス、ブーツ）
- **item_food.md** - 食料（ポーション効果付きも対応）

### ブロック系
- **block.md** - ブロック（通常、鉱石、装飾など）

---

## 🎯 使い方

### 1. 追加したいものに応じてテンプレートを選択

例：
- 新しい金属の剣を追加 → `item_tool.md`
- 新しい食べ物を追加 → `item_food.md`
- 新しい鉱石ブロックを追加 → `block.md`

### 2. テンプレートをコピーして記入

```bash
# 例: ミスリルの剣を追加する場合
cp .kiro/templates/item_tool.md .kiro/requests/mythril_sword.md
```

その後、`mythril_sword.md`を編集して必要な情報を記入

### 3. AIに実装を依頼

記入したファイルを添付して以下のように依頼：

```
.kiro/requests/mythril_sword.md の内容に基づいて、
アイテムを実装してください。
```

## 📋 各テンプレートの特徴

### 通常アイテム (item_normal.md)
**用途**: 素材、装飾品、特殊アイテム
**必要な情報**:
- スタック数
- クラフトレシピ
- 特殊機能（任意）

**実装されるファイル数**: 約4個

---

### ツール (item_tool.md)
**用途**: 剣、ピッケル、斧、シャベル、クワ
**必要な情報**:
- ツールティア（耐久値、採掘速度、攻撃力）
- 攻撃速度
- 修繕材料

**実装されるファイル数**: 約7個
**注意点**: ツールティアクラスが必要

---

### 防具 (item_armor.md)
**用途**: ヘルメット、チェスト、レギンス、ブーツ
**必要な情報**:
- 防御力、タフネス、ノックバック耐性
- 4部位すべてのクラフトレシピ
- 防具テクスチャ（layer_1, layer_2）

**実装されるファイル数**: 約15個（4部位分）
**注意点**: 
- 防具マテリアルクラスが必要
- テクスチャが2種類（レイヤー1と2）

---

### 食料 (item_food.md)
**用途**: 食べ物、飲み物
**必要な情報**:
- 満腹度回復量
- ポーション効果（任意）
- 食べる速度

**実装されるファイル数**: 約5個
**注意点**: ポーション効果は複数設定可能

---

### ブロック (block.md)
**用途**: 建築ブロック、鉱石、装飾ブロック
**必要な情報**:
- 硬度、爆発耐性
- 採掘ツール要件
- ドロップ設定
- ワールド生成（鉱石の場合）

**実装されるファイル数**: 約11個
**注意点**: 
- ブロックステートとモデルが必要
- 鉱石の場合はワールド生成設定も

---

## 💡 テンプレート記入のコツ

### 1. バニラを参考にする
テンプレート内に参考値を記載しているので、それを基準に調整

### 2. バランスを考慮
- ツール: ダイヤとネザライトの中間くらいが無難
- 防具: 同上
- 食料: ステーキ（8, 0.8）を基準に

### 3. 不要な項目は削除
任意項目で使わないものは「なし」と記入するか削除

### 4. 一度に複数追加する場合
セットで追加する場合（防具4点、ツール5点など）は、
1つのファイルにまとめて記載してもOK

## 🔄 テンプレートの更新

新しいアイテムタイプが必要になったら、
このディレクトリに追加してください：

```
.kiro/templates/
├── item_normal.md
├── item_tool.md
├── item_armor.md
├── item_food.md
├── block.md
└── [新しいテンプレート].md
```
