# モブ追加テンプレート

## 基本情報
```yaml
エンティティID: example_mob
エンティティ名（日本語）: サンプルモブ
エンティティ名（英語）: Example Mob
```

## モブタイプ
```yaml
モブタイプ: 敵対モブ  # 敵対モブ/中立モブ/友好モブ/ボス
飛行: No
```

## ステータス
```yaml
体力: 20.0  # ゾンビ=20, エンダードラゴン=200
攻撃力: 3.0  # ゾンビ=3
移動速度: 0.25  # ゾンビ=0.23
```

## サイズ
```yaml
幅: 0.6  # プレイヤー=0.6
高さ: 1.8  # プレイヤー=1.8
```

## 攻撃
```yaml
攻撃タイプ: 近接  # 近接/遠距離
特殊効果: なし  # 毒/炎上/スロー/なし
```

## スポーン
```yaml
スポーン: 自然  # 自然/スポーンエッグのみ
バイオーム: 全て
明るさ: 暗い場所  # 暗い場所/明るい場所/どこでも
```

## ドロップ
```yaml
通常ドロップ:
  - アイテム: rotten_flesh
    数量: 0-2
経験値: 5
```

## スポーンエッグ
```yaml
背景色: "#AAAAAA"
ドット色: "#333333"
```
