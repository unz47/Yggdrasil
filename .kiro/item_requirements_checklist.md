# Minecraftアイテム追加 - 要件チェックリスト

## 必須要件

### 1. 基本情報
- [ ] **アイテムID**: 英小文字とアンダースコア（例: `mythril_sword`）
- [ ] **アイテム名（日本語）**: ゲーム内表示名
- [ ] **アイテム名（英語）**: 英語表示名
- [ ] **説明文**: アイテムの説明（任意）

### 2. アイテムタイプ
- [ ] **カテゴリ**: 
  - 通常アイテム
  - ツール（剣/斧/ピッケル/シャベル/クワ）
  - 防具（ヘルメット/チェストプレート/レギンス/ブーツ）
  - 食料
  - ブロックアイテム
  - その他

### 3. アイテム特性

#### 基本特性
- [ ] **スタック可能**: Yes/No
- [ ] **最大スタック数**: 1-64（スタック可能な場合）
- [ ] **耐久値**: あり/なし
- [ ] **最大耐久値**: 数値（耐久値ありの場合）

#### ツール特性（ツールの場合）
- [ ] **採掘速度**: 数値
- [ ] **攻撃力**: 数値
- [ ] **採掘レベル**: 木/石/鉄/ダイヤ/ネザライト
- [ ] **適用ブロック**: 採掘可能なブロックタイプ

#### 防具特性（防具の場合）
- [ ] **防御力**: 数値
- [ ] **タフネス**: 数値
- [ ] **ノックバック耐性**: 数値

#### 食料特性（食料の場合）
- [ ] **回復量**: 数値（満腹度）
- [ ] **満腹度回復**: 数値
- [ ] **食べる時間**: 通常/速い
- [ ] **効果**: ポーション効果（任意）

### 4. クリエイティブタブ
- [ ] **タブ**: どのクリエイティブタブに配置するか
  - 建築ブロック
  - 装飾ブロック
  - レッドストーン
  - 輸送
  - 雑貨
  - 食料
  - ツール
  - 戦闘
  - 醸造
  - カスタムタブ

### 5. レシピ
- [ ] **クラフトレシピ**: あり/なし
- [ ] **レシピタイプ**: 
  - 作業台（有形/無形）
  - かまど
  - 溶鉱炉
  - 燻製器
  - 鍛冶台
  - その他
- [ ] **材料**: 必要な材料リスト
- [ ] **レシピパターン**: クラフトパターン（有形の場合）

### 6. テクスチャ
- [ ] **テクスチャファイル名**: `item_id.png`
- [ ] **テクスチャサイズ**: 16x16推奨
- [ ] **テクスチャの場所**: 既存/新規作成

### 7. 特殊機能（任意）
- [ ] **右クリック動作**: あり/なし
- [ ] **特殊能力**: 説明
- [ ] **エンチャント可能**: Yes/No
- [ ] **修繕可能**: Yes/No
- [ ] **燃料として使用可能**: Yes/No
- [ ] **燃焼時間**: 数値（燃料の場合）

### 8. ドロップ・入手方法
- [ ] **クラフト**: Yes/No
- [ ] **モブドロップ**: モブ名
- [ ] **ブロック破壊**: ブロック名
- [ ] **チェスト戦利品**: 場所
- [ ] **取引**: 村人タイプ

### 9. その他
- [ ] **希少度**: Common/Uncommon/Rare/Epic
- [ ] **発光**: Yes/No
- [ ] **3Dモデル**: 通常/カスタム
- [ ] **アニメーション**: あり/なし
