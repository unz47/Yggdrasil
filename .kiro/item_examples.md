# アイテム追加クイックスタート例

## 例1: シンプルな素材アイテム

```yaml
アイテムID: mythril_ingot
アイテム名（日本語）: ミスリルインゴット
アイテム名（英語）: Mythril Ingot
カテゴリ: 通常アイテム
スタック可能: Yes
最大スタック数: 64
クリエイティブタブ: 雑貨
クラフトレシピ: あり（かまど）
材料: mythril_ore
```

## 例2: カスタムツール

```yaml
アイテムID: mythril_sword
アイテム名（日本語）: ミスリルの剣
アイテム名（英語）: Mythril Sword
カテゴリ: ツール
ツールタイプ: 剣
攻撃力: 8.0
攻撃速度: -2.4
耐久値: 1500
クリエイティブタブ: 戦闘
クラフトレシピ: あり（作業台・有形）
パターン:
  _M_
  _M_
  _S_
  M=mythril_ingot, S=stick
```
